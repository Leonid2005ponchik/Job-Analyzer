# Пример вычисления предсказания зарплаты с точки зрения чистой математики 

## Исходные данные 
1. 35000 - 34000 - вакансия №1
2. 40000 - 41000 - вакансия №2
3. 38000 - 37000 - вакансия №3
4. 42000 - 43000 - вакансия №4
5. 36000 - 35000 - вакансия №5

## Вычисление функции потерь 
1. Пример: (1/n) · Σ(1000)² = 1 000 000
2. Пример: (1/n) · Σ(-1000)² = 1 000 000
3. Пример: (1/n) · Σ(1000)² = 1 000 000 
4. Пример: (1/n) · Σ(-1000)² = 1 000 000
5. Пример: (1/n) · Σ(1000)² = 1 000 000 

## Частная производная - расчет градиентов 
1. ∂L/∂ŷ_i = 1000 
2. ∂L/∂ŷ_i = -1000
3. ∂L/∂ŷ_i =  1000
4. ∂L/∂ŷ_i = -1000
5. ∂L/∂ŷ_i = 1000

## Оформляем градиенты в виде матрицы 

- Градиент  = ∇L(1000, -1000, 1000, -1000, 1000) 

## Вычисляем градиентный спуск 

- **Формула:** ŷ_i^(new) = ŷ_i^(old) - η · ∂L/∂ŷ_i 
1. Пример 1 для вакансии №1: ŷ_i^{(new)} = ŷ_i^{(old)} - 0,1 · 1000 =  34000 - 0,1 · 1000 = 34000 - 100 = 33900
2. Пример 2 для вакансии №2: ŷ_i^{(new)} = ŷ_i^{(old)} - 0,1 · (-1000) =  41000 + 0,1 · 1000 = 41000 + 100 = 41100
3. Пример 3 для вакансии №3: ŷ_i^{(new)} = ŷ_i^{(old)} - 0,1 · 1000 =  37000 - 0,1 · 1000 = 37000 - 100 = 36900
4. Пример 4 для вакансии №4: ŷ_i^{(new)} = ŷ_i^{(old)} - 0,1 · (-1000) =  43000 + 0,1 · 1000 = 43000 + 100 = 43100
5. Пример 5 для вакансии №5: ŷ_i^{(new)} = ŷ_i^{(old)} - 0,1 · 1000 =  35000 - 0,1 · 1000 = 35000 - 100 = 34900

## Новое значение функции потерь для следующего шага 

- Новая MSE: (1100² + 1100² + 1100² + 1100² + 1100²) / 5 = 1 210 000

1. Пример: (1/n) · Σ(35000 - 33900)² = (1/n) · Σ(1100)² = 1 210 000 
2. Пример: (1/n) · Σ(40000 - 41100)² = (1/n) · Σ(-1100)² = 1 210 000
3. Пример: (1/n) · Σ(38000 - 36900)² = (1/n) · Σ(1100)² = 1 210 000
4. Пример: (1/n) · Σ(42000 - 43100)² = (1/n) · Σ(-1100)² = 1 210 000
5. Пример: (1/n) · Σ(36000 - 34900)² = (1/n) · Σ(1100)² = 1 210 000

## Частная производная - расчет градиентов

## Разложение в ряд Тейлора для приближения функции потерь 

- **Важно:** - функция нужна для аппроксимирования линейной функции потерь. Разница большая, т.к дает линейное преображение функции, а сама функция является квадратичной. Нужен 2-й порядок для точности.  

- **Формула:** - (ŷ + ε) ≈ L(ŷ) + ε · L'(ŷ)
- **Уточненение для "ε":** - ε = -η · ∂L/∂ŷ₁

1. Пример: (34000- 110) =  1 000 000 + (-0,1 * 1000) * 1000 = 900 000 
2. Пример: (41000 + 110) = 1 000 000 + (-0,1 * -1000) * (-1000)= 900 000 
3. Пример: (37000 -110) =  1 000 000   + (-0,1 * 1000) * 1000 =  900 000 
4. Пример: (43000 +  110) =  1 000 000  + (-0,1 * - 1000) * (-1000) = 900 000 
5. Пример: (35000 - 110) = 1 000 000 + (-0,1 * 1000) * 1000 = 900 000 

## Метод Ньютона 

**Важно:** метод при котором сходимость быстрее 
**Необходимо:** 
1. Взять вторую производную (названо в честь Гессиана)
2. Вторая производная функции L" = 2/n - остается только константа 
3. Сам метод Ньютона - ŷ_new = ŷ - L'/L"

1. Пример 1 для вакансии №1: 
    - L' = (2/5) * (34000 - 35000) = 0.4 * (-1000) = -400 (n = 5 по количеству вакансий)
    - L" = 2/5 = 0,4 (Гессиан)
    - ŷ_new = 34000 - (-400)/0,4 = 35000 (метод Ньютона)
2. Пример 2 для вакансии №2: 
    - L' = (2/5) * (41000 - 40000) = 0.4 * (1000) = 400 
    - L" = 2/5 = 0,4 
    - ŷ_new = 41000 - 400/0,4 = 40000
3. Пример 3 для вакансии №3:
    - L' = (2/5) * (37000 - 38000) = 0.4 * (-1000) = -400 
    - L" = 2/5 = 0,4 
    - ŷ_new = 37000 - (-400)/0,4 = 38000  
4. Пример 4 для вакансии №4:
    - L' = (2/5) * (43000 - 42000) = 0.4 * (1000) = 400 
    - L" = 2/5 = 0,4 
    - ŷ_new = 43000 - 400/0,4 = 42000
5. Пример 5 для вакансии №5: 
    - L' = (2/5) * (35000 - 36000) = 0.4 * (-1000) = -400 
    - L" = 2/5 = 0,4 
    - ŷ_new = 35000 - (-400)/0,4 = 36000 


## L2 - регуляризация (Ridge regression)
**Важно:** - функция, которая предотвращает переобучение функции через штрафование больших весов. 

**Формула:** ∂L_ridge / ∂ŷ_i = MSE + λ*Σŷ_i^2
- λ - коэффициент регуляризации. Чем выше, тем модель прощещ, чем ниже, тем сложнее. 
Берем значение = 0,01
- Берем производную и раскладываем MSE: L2' = (2/n)(ŷ_i - y_i) + 2λŷ_i

- Пример для вакансии №1: ∂L_ridge / ∂ŷ_i = (2/5)(34000 - 35000) + 2 * 0,01 * 34000 = 280 
- Пример для вакансии №2: ∂L_ridge / ∂ŷ_i = (2/5)(41000 - 41000) + 2 * 0,01 * 41000 = 820
- Пример для вакансии №3: ∂L_ridge / ∂ŷ_i = (2/5)(37000 - 38000) + 2 * 0,01 * 37000 = 340
- Пример для вакансии №4: ∂L_ridge / ∂ŷ_i = (2/5)(43000 - 42000) + 2 * 0,01 * 43000 = 1260
- Пример для вакансии №5: ∂L_ridge / ∂ŷ_i = (2/5)(36000 - 35000) + 2 * 0,01 * 36000 = 1120


## Градиентный бустинг и остатки 
**Важно:** - остаток пропорционален отрицательному градиенту MSE, но функция остатков показывает следующему дереву, в какую сторону и как сильно нужно двигаться, чтобы ошибаться меньше.
**Дополнительная информация:** 
- Пример выше с методом Ньютона был включен с целью показать разницу в логистической регрессии и построение деревьев решений у градиентного бустинга. 
**Формула остатков:** r_i = y_i - ŷ_i
**Формула отрицательного градиента MSE:** -∂L / ∂ŷ_i = (2/n) * r_i

- Пример для вакансии №1: 
    - r_i = 35000 - 34000 = 1000 
    - -∂L / ∂ŷ_i = (2/5) * 1000 = 400
- Пример для вакансии №2: 
    - r_i = 40000 - 41000 = -1000 
    - -∂L / ∂ŷ_i = (2/5) * -1000 = -400
- Пример для вакансии №3: 
    - r_i = 38000 - 37000 = 1000 
    - -∂L / ∂ŷ_i = (2/5) * 1000 = 400
- Пример для вакансии №4: 
    - r_i = 42000 - 43000 = -1000 
    - -∂L / ∂ŷ_i = (2/5) * -1000 = -400
- Пример для вакансии №5: 
    - r_i = 36000 - 35000 = 1000 
    - -∂L / ∂ŷ_i = (2/5) * 1000 = 400


## Доверительный интервал для предсказания 
- Стандартное отклонение из частной производной расчета градиентов ≈ 1100
